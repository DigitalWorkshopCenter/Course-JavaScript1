Summary

In this lesson, we will apply everything that we've learned up to this point and create a simple JavaScript web application. You may choose an application of your liking from the list below. Alternatively, if you have an idea that will satisfy the requirements for this assignment, you may request authorization from your instructor.

Requirements for Project

This project is intended to test your understanding of the course material up to this point, and give you an opportunity to demonstrate your learning, stretch your understanding, and help you progress in your skills as a web developer. Your web application should:

  • Be useful in nature - something that people would want to use.

  • Include many different variables

  • Incorporate multiple JavaScript functions

  • Include some type of loop

  • Have the ability to update data on the webpage (DOM) from JavaScript

  • Include an HTML form with validation that calls a JavaScript function
