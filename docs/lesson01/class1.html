Overview

The focus of this assignment is to dive right in to writing JavaScript. Here are our goals for today:

Computer Setup

VS Code

To set up our computers to write JavaScript, all we need to do is download a text editor. We will be using Visual Studio Code also known as VS Code.

  1. Open this webpage in your browser.

  2. Download VS Code for your operating system.

  3. Install VS Code.

  4. Accept all the default settings.

Now that it is installed. Open VS Code. We are going to download an extension for it that will allow us to run/execute JavaScript here in VS Code. Click on the Extensions Icon and search code runner, then click install.

Hello World!

"Hello, World!" is a time-honored program that outputs or displays the message "Hello, World!". It is used to illustrate the basic syntax of a programming language and because it is very simple, it is often the first program written by people learning to code. Our first program today will be our "Hello, World!" program.
