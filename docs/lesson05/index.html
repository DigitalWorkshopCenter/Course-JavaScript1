Summary

In this lesson, we will apply everything that we've learned up to this point and create an advanced JavaScript web application. You may choose an application of your liking from the list below. Alternatively, if you have an idea that will satisfy the requirements for this assignment, you may request authorization from your instructor.

Requirements for Assignment

This assignment is intended to build upon your understanding and experience gained from your previous web application, and give you an opportunity to demonstrate your learning, stretch your understanding, and help you progress in your skills as a web developer. This advanced web application should:

  • Be useful in nature - something that people would want to use.

  • Incorporate multiple JavaScript functions

  • Dynamically generate content on the webpage (DOM) from JavaScript in several ways

  • Include an HTML form with validation that calls a JavaScript function

  • Create at least 2 custom events
