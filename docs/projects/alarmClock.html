Summary

A company is hiring you to build them an alarm clock application. They have tried grandfather clocks, wrist watches, vibrating watches, digital alarm clocks, random mobile apps and bedside assistants like Google Home and Alexa...none of these have done everything that they wanted and needed an alarm clock to do. You have been tasked to build the alarm clock of the century and they will pay you well for it.

Alarm Clock Feature List

The application will is required to have a home.html page with a js and css file for it. But the company wants to ensure that it is ready to add more functionality, so you should design the application file structure planning to add more pages.

  • Home page

  • Shows current time (updates as seconds pass)

  • Shows fun image that should motivate someone to wake up (loaded from JavaScript) (stretch, this changes when alarm goes off)

  • Shows dynamic background color (loaded from JavaScript)

  • Shows all current alarms that are set

  • Includes a form to add new alarms

  • Add validation to the form to ensure that only alarms are added if they have a unique title and a time.

  • When alarm goes off, show something different on the page and play a sound. Have a button to turn off the alarm.

Eventually, these pages may be added (stretch challenges)

  • Theme Preferences page

    • Allow user to change the background color of the application

    • Allow user to change the image on the home page

  • Share page

    • Has a simple link to send an email containing this website in the body of the email

    • Stretch - add another way to share

  • One of your choosing (Ideas include, but are not limited to)

    • Log of when alarms had been set

    • Login (this does not have to be fully functional)
