Investment Calculator Project

Summary

Many bank accounts and other investments pay compound interest on the money that is saved in the account. This means each time the bank pays interest, the interest is added to the original money, which is called the principal. Then the next time the banks pays interest, it pays interest on the principal and the previous interest. The balance of such an account and its compound interest can easily be computed using a loop.

Investment Calculator Feature List

The application is required to have a home.html page with a js and css file for it. But the company wants to ensure that it is ready to add more functionality, so you should design the application file structure planning to add more pages.

  • Has professional look and feel

  • Includes an html form with the following fields:

    • Balance of the account (amount of money at the beginning)

    • The annual interest rate

    • Duration of investment (in months)

  • A button to submit the form

  • Implements the following functions:

    • submitData(). This function will retrieve the form data, and check for validation errors (the three numbers need to be present and not less than 0). If there is an error, display the error by calling the updateDOM function and stop the remaining code execution. If there is no error, call the computeFutureValue function.

    • updateDOM(value, color). This function will display the error message when there is a validation error, and it will also display the future value if all the code execution goes well. The value parameter will either be the error message string, or the future value number string. The color you can use to make the error red, or the future value green, etc.

    • computeFutureValue(balance, annualRate, months). This function will include the loop and the logic to add each month's interest on to the balance to compute the future value. After the value has been computed, call updateDOM function.

Stretch Challenges

  • Find a professional image that reminds you of investments to use as the body background.

  • Add commas to the output value number if it is more than $999 (for example, $1,000.00 and $3,123,957.45)
