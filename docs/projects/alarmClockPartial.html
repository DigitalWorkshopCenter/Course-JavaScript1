Alarm Clock - Partially Completed Web Application

A company is hiring you to build them an alarm clock application. They have tried grandfather clocks, wrist watches, vibrating watches, digital alarm clocks, random mobile apps and bedside assistants like Google Home and Alexa...none of these have done everything that they wanted and needed an alarm clock to do. They had someone else come in and do it, but they always wanted to sleep so badly, the application is dark and played sleep music when alarms went off. Now, alarms don't even go off! You are being hired to make this alarm clock great again.

Alarm Clock Feature List

The application will is required to have a home.html page with a js and css file for it. But the company wants to ensure that it is ready to add more functionality, so you should design the application file structure planning to add more pages.

  • Home page

  • Shows current time (updates as seconds pass)

  • Shows fun image that should motivate someone to wake up (loaded from JavaScript) (stretch, this changes when alarm goes off)

  • Shows dynamic background color (loaded from JavaScript) that is bright to help people wake up

  • Shows all current alarms that are set

  • Includes a form to add new alarms

  • Add validation to the form to ensure that only alarms are added if they have a unique title and a time.

  • When alarm goes off, show something different on the page and play a sound (An energetic sound). Have a button to turn off the alarm.

  • Several comments were left throughout the application by the previous developer regarding things that need to get added in order to complete this project.
