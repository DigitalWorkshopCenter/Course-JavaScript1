Hands-on Exercise 1
Changing page data with the DOM

Overview

The focus of this activity is to learn how to manipulate the DOM (AKA: change data on the page from JavaScript). Here are our goals for today:

  1. Create an html file and a JavaScript file.

  2. Provide a button for the user to click.

  3. When the user clicks this button, call a function in your js file.

  4. When this function is called, change the background color of the webpage.

Stretch

  • Create a colors array and randomly assign a color to the background of the page when the button is clicked.
