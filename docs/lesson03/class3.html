Hands-on Exercise 3
Using Forms

Overview

The focus of this activity is to learn how to manipulate the DOM, use forms and trigger an event. Here are our goals for today:

  1. Create an html file and a JavaScript file.

  2. Create a form in the html file with several input boxes for the user

  3. When the user clicks the submit button, trigger an event to call a function in your js file.

  4. This function will then add data to a list on the page for the user to see.
