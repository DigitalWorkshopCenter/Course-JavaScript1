Lesson Assignment

Overview

The assignment for this lesson is intended to strengthen your ability to work with the DOM, events and forms. Your tasks are as follows:

  1. Create a fun web page of your choosing

  2. Include a form with an input box for the user to put the url of an image on the web

  3. When they submit the form (by pressing the button) it will call a function

  4. This function will then create an img element an put it on the page with the new image.

Stretch Challenges

  • Add form validation to ensure that the user put a url into the input box

  • Add some way to count how many images have been added, and display that somewhere else on the page.

  • Along with each image that appears in the page, also create a button that could delete that image from the page.
