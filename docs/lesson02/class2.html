Hands-on Exercise 2
Practice with Loops

Overview

The focus of this activity is to get comfortable using loops in JavaScript. Here are our goals for today:

  • Write several programs that use repetition

  • Try out several types of loop

Programs

You may do the following in the browser with an html file that uses a js file, or just in the terminal in vs code with a js file.

While Loop

Create a program that displays the numbers 1 to 100 using a while loop.

For Loop

Create a program that displays the numbers 1 to 100 using a for loop.

For Loop

Create a program that displays the EVEN numbers from 1 to 100 using a for loop.

For Each Loop

Create a program with an array that contains 10 names. Write a loop to display each of those names to the user.
