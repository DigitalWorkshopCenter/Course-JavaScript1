Hands-on Exercise 1
Exploring JavaScript Data Structures

Overview

The focus of this activity is to get comfortable declaring variables of various types and knowing when to use what type. Here are our goals for today:

  1. Declare variables of each data type

  2. Display this data in the terminal in VS Code.

  3. Display this data in the browser console.

  4. Gain practice thinking about what data type is best for certain situations.

Declaring Variables

  1. Create a js file. Each on its own line of code, declare new variables of the following data types by assigning a value to it of that type.

    • string

    • number

    • object

    • array

    • function

  2. Log each of these to the console.

  3. Use the Run button like we did in the previous lesson to see the output in the terminal in VS Code.

  4. Having written code to declare and log these variables, now write code to display them in the console of the browser. HINT (you will need to create an html file, link the js file to it, and open it with Live Server).

Variable Quiz

Programmers frequently need to solve difficult problems. Having the understanding to know when to use certain data structures will benefit you as a programmer practically every day that you work. Answer the following questions to the best of your ability regarding JavaScript data structures:

  1. How would I store my mailing address in a variable? What datatype would it be?

  2. How would I store my age in a varieble? What datatype would it be?

  3. How would I store a list with all the names of the students in this class? What datatype would it be?

  4. How could I store the make, model, color and year of a car? What datatype would it be?

  5. How could I store a way to display the current time to the user? What datatype would it be?

  6. What does the typeof operator do in JavaScript?

  7. What does the following expression return? 

    
    typeof(3);

  8. What does the following expression return? 

    
    typeof(3) === typeof(4.32);

  9. What is the value of the hello variable in the following expression? 

    
    let hello;
