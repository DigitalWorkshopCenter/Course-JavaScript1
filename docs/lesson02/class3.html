Hands-on Exercise 3
Programming with Functions

Overview

The focus of this activity is to get comfortable writing functions in JavaScript. Here are our goals for today:

  • Write a bunch of functions

  • Understand the value of functions

  • Learn how to use the browser debugger

House Volume Program

The following program is to be run in the browser. You will create an html file (which you will open with Live Server) and a js file (referenced in the html file) which will contain code fulfilling the requirements listed below.

  • Create a function: getDepth() that will prompt the user for the width of a house and return that value as a number.

  • Create a function: getWidth() that will prompt the user for the depth of a house and return that value as a number.

  • Create a function: getHeight() that will prompt the user for the height of a house and return that value as a number.

  • Create a function: getSweep() that will prompt the user for the sweep (roof side length) of a house and return that value as a number.

  • Create a function: houseVolume() that will

    • take 4 parameters (width, depth, height, sweep)

    • call the livingVolume function

    • call the roofVolume function

    • compute the house volume by adding the living volume and the roof volume

    • return the volume of the house

  • Create a function: livingVolume() that will take three parameters (width, depth, height), compute the volume of the living space and return it.

  • Create a function: roofVolume() that will take three parameters (width, depth, sweep), call the triangleArea function, and compute and return the volume of the roof.

  • Create a function: triangleArea() that will take three parameters (a, b, c) - the three sides of the triangle, and return the area of the triangle. According to Heron's Formula, the way you compute the area of a triangle given three sides is: Area = √p(p−a)(p−b)(p−c)

  • Having created all of these functions, write code to call each of the get functions and then compute and display the house volume to the user.
